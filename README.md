# Desafio Dextra

## Instalação

Clone o projeto
```
git clone https://github.com/marchioreto/dextra-challenge.git
```

Usando o Maven

execute:
```
mvn clean install
```
Após a conclusão da instalação, entrar na pasta **/target**

execute:
```
java -jar challenge-0.0.1-SNAPSHOT.jar
```
acesse o sistema pelo endereço:

http://localhost:8080
